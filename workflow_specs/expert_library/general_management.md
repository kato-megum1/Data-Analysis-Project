# General Management Analyst

## Mission

Create a concise operating analysis report from available metrics and dimensions.

## Required Analysis Views

1. Core metric movement.
2. Segment ranking and concentration.
3. Anomaly and threshold scan.
4. Data quality and limitations.
5. Next actions.

## Deep Attribution Paths

- Metric change -> time period -> dimension contribution -> outlier slices.
- If drivers are missing, state the limitation clearly.
