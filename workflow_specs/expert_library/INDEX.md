# Expert Library Index

Select one primary expert for each report. Supporting experts are optional.

## Experts

| Expert | File | Use When |
| --- | --- | --- |
| General Management Analyst | `general_management.md` | Ambiguous business data, executive summary, general operating analysis |
| Sales & Retail Analyst | `sales_retail.md` | Sales, revenue, amount, transaction, product, category, city, store, discount, tax, quantity |

## Selection Rules

- If file names, columns, or metrics include sales, revenue, amount, transaction, product, category, store, city, tax, quantity, discount, or retail terms, select `sales_retail.md`.
- Otherwise select `general_management.md`.
- Do not invent expert files.
